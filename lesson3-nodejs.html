Install Node.js & HAXTheWeb

Node.js is a JavaScript runtime that lets you run JavaScript outside the browser. We'll install it and then set up HAXTheWeb â€” a powerful tool for creating modern web experiences.

Part 1 · Install Node.js

macOS

Use Homebrew to install Node.js:

zsh 
brew install node

Verify Node.js installation:

zsh 
node --version
npm --version

You should see version numbers like v20.x.x and 10.x.x

Linux

On Ubuntu / Debian, use NodeSource:

bash 
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

Check installation:

bash 
node --version
npm --version

You should see version numbers like v20.x.x and 10.x.x

Windows

Download Node.js from nodejs.org

  • Click the LTS (Long Term Support) version
  • Run the downloaded .msi installer
  • Accept defaults and click Next through the wizard
  • Check "Automatically install necessary tools" if prompted
  • Click Finish

Open PowerShell or Git Bash and verify:

PowerShell 
node --version
npm --version

You should see version numbers like v20.x.x and 10.x.x

Part 2 · Install HAXTheWeb

H

What is HAXTheWeb?

HAXTheWeb is a powerful web authoring system that makes it easy to create modern, accessible websites and web experiences. It provides a visual editor with drag-and-drop capabilities while generating clean, standards-based HTML.

The @haxtheweb/create tool scaffolds new HAX projects with best practices built-in.

Install HAXTheWeb globally (all platforms)

This command works the same on macOS, Linux, and Windows. Open your terminal and run:

terminal 
npm install --global @haxtheweb/create

Installation Note

This might take a minute or two as npm downloads and installs the HAXTheWeb tools. You'll see progress indicators in your terminal.

Verify the installation

Check that HAXTheWeb is installed correctly:

terminal 
npm list -g @haxtheweb/create

You should see output showing the installed version of @haxtheweb/create

Success!

You now have Node.js, npm, and HAXTheWeb installed. You're ready to create modern web projects!
