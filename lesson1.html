Git

Git is a version control system. It keeps a timeline of your project on your own computer. You can go back in time, or branch off and try ideas safely.

GitHub

GitHub is where you put your Git project online. Now teammates can see it, suggest changes, comment, and contribute.

Visual Studio Code

VS Code is your workspace. You write code here, preview, commit, and push to GitHub directly.
