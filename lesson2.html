Lesson 2: Install Git & Visual Studio Code

We'll install Git and VS Code on macOS, Linux, and Windows. Then we'll confirm everything works.

Part 1 ‚Äî Install Git

macOS

Use Homebrew to install Git: 

brew install git

Check it worked: 

git --version

Linux

On Ubuntu / Debian: 

sudo apt-get install git

Verify: 

git --version

Windows

Download Git for Windows from git-scm.com, run the installer.

Then open Git Bash and run: 

git --version

Part 2 ‚Äî Install Visual Studio Code

macOS

Download from code.visualstudio.com/download.

Verify:

code --version

Linux

Install via Snap: 

sudo snap install code --classic

Then:

code --version

Windows

Download for Windows from code.visualstudio.com/download, run installer.

Then: 

code --version

Git is installed. VS Code is installed. Your machine is ready. Next we link your computer to GitHub so you can push code.
