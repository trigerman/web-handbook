Lesson 2: Install Git & Visual Studio Code

We'll install Git and VS Code on macOS, Linux, and Windows. Then we'll confirm everything works so you can ship code with confidence.

Part 1 · Install Git

macOS

Use Homebrew to install Git:

zsh 
brew install git

Check it worked:

zsh 
git --version

Linux

On Ubuntu / Debian:

bash 
sudo apt-get install git

Verify:

bash 
git --version

Windows

Download Git for Windows from git-scm.com and run the installer.

Then open Git Bash and run:

Git Bash 
git --version

Part 2 · Install Visual Studio Code

macOS

Download VS Code from code.visualstudio.com.

Verify the command line launcher:

zsh 
code --version

Linux

Install via Snap:

bash 
sudo snap install code --classic

Then confirm:

bash 
code --version

Windows

Download the Windows build from code.visualstudio.com and run the installer.

Verify from Git Bash or PowerShell:

shell 
code --version

You're ready to build

Git is installed. VS Code is installed. Your machine is primed for real projects. Next we'll connect everything to GitHub so you can push, pull, and collaborate.
